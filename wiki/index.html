Serverinfo
Der Minecraft Spigot Server ist auf Witherhosting.com gehostet und 24/7 durchgehend In-Game erreichbar. (*)

Aktuell läuft der Server im Paket MultiServer Professional 4GB auf einem deutschen Server, was für einen kleinen privaten Minecraft-Server wie diesen völlig ausreichen sollte - auch mit den zahlreichen installierten Plugins.

Auch wenn der Spigot Server auf der Java Edition läuft, ist es dank zweier Plugins problemlos möglich, dass auch Spieler der Bedrock Edition (Windows 10, XBox, Android, iOS) darauf spielen können.

(*außer bei Wartungsarbeiten oder Plugin-Tests)
